
# Autism Spectrum Disorder Detection

A machine learning project to assist in the early detection of Autism Spectrum Disorder (ASD) based on behavioral and demographic features.

---

## Project Description

This project applies machine learning techniques to predict the likelihood of Autism Spectrum Disorder in individuals. By analyzing patterns in behavioral questionnaire responses and demographic data, the model aims to support early screening and assist healthcare professionals.

Key goals:
- Automate ASD screening using ML classification models
- Identify the most significant features contributing to ASD prediction
- Provide an accurate and interpretable prediction pipeline

---

## Technologies Used

- **Language:** Python 3.x
- **Libraries:** scikit-learn, pandas, numpy, matplotlib, seaborn
- **Environment:** Jupyter Notebook / VS Code
